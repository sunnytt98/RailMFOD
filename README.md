# RailMFOD
A dataset developed for foreign object segmentation in open railway scenarios.

# Download
The dataset can be downloaded from https://pan.baidu.com/s/1Hie9F8RDcJZ_aOnj92Zq5A  kv8g 

# Note 
The LiDAR sensors used different scanning modes during the collection of the training and test sets. The test set contains ROI (Region of Interest) areas, resulting in denser point clouds in distant regions.
If you have any questions or encounter any issues while using the dataset, please feel free to contact us.
